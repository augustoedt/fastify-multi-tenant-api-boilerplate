# Fastify Boilerplate

Current state its almost a full copy of https://github.com/TomDoesTech/user-api, credits to @TomDoesTech.

 - Multi Tenant Role API
 - Create Application
 - Register User
 - Login
 - Create a role
 - Assign role to User

### Running local database

- Must have Docker installed

```bash
docker-compose up -d
```
